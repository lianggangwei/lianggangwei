# 自动化工具包

一套完整的Python自动化工具，涵盖网页自动化、数据处理、AI集成、系统工具等多个领域。

## 目录结构

```
tools/
├── web/              # 网页自动化工具
│   ├── scraper.py    # 网络爬虫
│   └── browser.py    # 浏览器自动化
├── data/             # 数据处理工具
│   ├── excel.py      # Excel处理
│   ├── csv_tool.py   # CSV处理
│   └── database.py   # 数据库操作
├── ai/               # AI工具
│   ├── api_client.py # AI API集成
│   └── local_llm.py  # 本地模型支持
├── system/           # 系统工具
│   ├── file_ops.py   # 文件操作
│   └── backup.py     # 备份工具
└── examples/         # 示例脚本
```

## 安装依赖

```bash
pip install -r tools/requirements.txt
```

## 快速开始

查看 `tools/examples/` 目录下的示例脚本开始使用。
